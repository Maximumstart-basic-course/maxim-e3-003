Что такое EcmaScript?

EcmaScript - встраиваемый расширяемыый не имеющий средств ввода-вывода язык программирования, используемый в качестве основы для построения других скриптовых языков.

Для чего используются и чем отличаются атрибуты async и defer?

Данные атрибуты используются для сопряжения загрузки страницы и скриптов. Async позволяет загружать скрипт в фоновом режиме без блокировки загрузки основного контента. Скрипт выполнится, когда будет загружен. Defer работает так же, но скрипт выполнен после полной загрузки страницы, и, к тому же, сохранится очередность выполнения.

Почему предпочтительнее подключать скрипты перед body, а не в head?

Таким образом, скрипт загружается быстрее.

Какие основные отличия интерпретируемого языка программирования от компилируемого?

Компилируемый язык программирования подразумевает преобразование программного кода в машинный, чем занимается специальная программа-компиллятор. В интерпретируемом языке программирования используется программа-интерпретатор, время преобразования кода которой занимает значительно больше времени.

Что такое языки высокого и низкого уровня? Какие ты знаешь?

Язык высокого уровня - это язык программирования, подразумевающий введение смысловых конструкций, описание которых на машинном коде очень длинны и сложны для понимания. К таким языкам относятся С, С++, Delphi, Java, JavScript, PHP, Pascal. Язык низкого уровня является близким к форме, которую компьютер понимает непосредственно, т.е., к машинному коду. К ним относится Assembler.

Для чего используются переменные? Какие ключевые слова используются для создания переменных? Какие есть отличия в переопределении значений при задании переменной через let, const и var?

Переменные используются для записи значения с целью дальнейшего использования. Для создания переменных используются ключевые слова var, let и const. Const используется в тех случаях, когда значение переменной не будет изменяться. Var и let применяют в обратном случае.

Может ли имя переменной начинаться с цифры, символов _ или $?

Имя переменной может начинаться только с символа $.

Какие слова запрещено использовать в названии переменной?

В названии переменной запрещается использовать зарезервированные слова, а именно abstract, boolean, break, byte, case, catch, char, class, const, continue, debugger, default, delete, do, double, else, enum, export, extends, false, final, finally, float, for, function, goto, if, implements, import, in, instanceof, int, inteface, long, native, new, null, package, private, protected, public, return, short, static, super, switch, synchronized, this, throw, throws, transient, true, try, typeof, var, volatile, void, while, with.

Что такое camelCase и snake_case?

camelCase - "верблюжья нотация", её пример - запись переменной как "borderLeftWidth". snake_case - стиль написания имени переменной с использованием нижнего подчеркивания.

Что такое "типы данных"? Какой оператор позволяет определить тип данных?

Типы данных используются для возможности классификации всех используемых данных. Они разделяются на простые и сложные. К простым относятся примитивные (число/number, строка/string, null, undefined, логические/boolean, символ/symbol) и сложные (массив/arrays, объект/object, функция/function). Тип данных позволяет определить оператор typeof.

Привести строчное значение к числу можно, добавив перед строкой знак "+", можно также добавить "\n".

Чем отличаются операторы "==" и "==="?

Оператор "==" используется при сравнении значений, но не типов данных. Оператор "===" - для сравнения значений и типов данных.

Какие преимущества дают обратные кавычки (`) при указании строки?

Обратные кавычки позволяют переносить значение на другие строки.

Что такое "алгоритм"? Какие ты знаешь типы алгоритмов?

Алгоритм - набор инструкций, описывающий порядок действий исполнителя для достижения некоторого результата. Алгоритмы бывают линейными, разветвленными и циклическими.

Зачем нужны циклы? Какие виды циклов ты знаешь? В чем их основные отличия?

Циклы нужны в случае необходимости многократного повторения какого-либо действия. Циклы бывают: while (проверяет условие, и если оно приравнивается к true, то тело цикла выполняется), do...while (в нем сначала выполняется тело цикла, а потом проверяется соответствие условию), бесконечный цикл (infinite loop), цикл for (отличается компактностью).

Что делают операторы ++i и i++ и в чем их отличие?

Данные операторы увеличивают i на единицу. Их отличие заключается в том, что i++, в отличие от i++, возвращает значение, которое было до увеличения.

Что такое "итерация"?

Итерация - повторение цикла.

Как называются и чем отличаются операторы || и &&?

|| (или) возвращает первую из двух переменных, если она может быть преобразована к true, в ином случае, возвращает вторую. && (и) возвращает первую из двух переменных, если она может быть преобразована к false, в ином случае, возвращает вторую.

Для чего используется break? Почему в конструкции switch нужно использовать break для каждого case?

break завершает текущий цикл или конструкции switch. Если break нет, то выполнение цикла пойдет по последующим case, минуя остальные проверки.

Что такое область видимости?

Область видимости - часть контекста выполнения, в котором эта переменная существует.

Чем отличается Function Declaration от Function Expression?

Function Declaration - функция, объявленная в основном потоке кода (function sum(a, b) {return a + b;}). Function Expression - объявление функции в контексте какого-либо выражения (var sum = function(a, b) {return a + b;}).

Что такое замыкание?

Замыкание - комбинация функции и лексического окружения, в котором эта функция была определена.
